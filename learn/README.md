# Learn 学习执行层

## 定位
`learn/` 是 GetMyPage 项目的学习执行层，用于把“学习 Spring Boot + 落地博客功能”转成可跟踪、可验收、可复盘的工程过程。

本目录与 `docs/` 的关系：
- `docs/`：偏稳定规范与全局说明。
- `learn/`：偏过程实践与阶段产出。

## 使用方式
1. 从 [INDEX.md](./INDEX.md) 选择当前 Step。
2. 阅读该 Step 的 `00-overview.md` 明确目标与 DoD。
3. 在 `01-task-board.md` 勾选任务并执行。
4. 严格按 `02-acceptance.md` 运行命令验收。
5. 在 `05-review.md` 完成复盘，并更新下一步行动。

## 统一约束
- 每个 Step 必须包含：学习目标、完成定义（DoD）、执行命令、验收结果、架构决策记录（ADR-Style）、下一步。
- 每个 Step 必须包含可执行命令验收，不接受“仅口头完成”。
- 每个 Step 必须包含扩展学习（性能/一致性/安全）与架构对比思考。

## 推荐节奏
- 单个 Step 建议 5-10 小时。
- 每次开发结束，至少更新一次任务板与复盘。
- 关键接口变更后，同步回写 `docs/` 正式文档。
